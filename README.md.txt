na poczatku dokonywalem commitów ze zlym configiem.
Jak juz sie zorietnowalem zmienilem na swoje dane/ 
git init inicjalizuje nasze nowe repozytorium 
git add dodaje dany plik na którym polcenie zostalo wykonane do naszego repo
git commit commituje dany plik po wczesniejszym dodaniu flaga -m pozwala na dodanie wiadomosci co zmieniano w danym commicie 
git remote add origin https://github.com/slawick12/git.git - dodaje zdalne repo do serwisu github 
git push -u origin master puszuje nasze commity na mastera git status pozwala sprawdzenia statusu plików 
git config zmienia np autora commitow koniec
